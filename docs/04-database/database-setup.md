# Database Setup – Marucha Mobile App

This document explains how to create and populate the **Marucha Mobile App** database in a local development environment.

---

# 1. Database Engine

The project uses:

- **MySQL**
- Local development environment with **XAMPP**
- Database administration through **MySQL Workbench**

---
