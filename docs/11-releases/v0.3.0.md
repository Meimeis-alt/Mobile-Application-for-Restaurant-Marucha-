# Release v0.3.0 — PHP Backend Base

## Resumen
La versión **v0.3.0** corresponde a la implementación de la base funcional del backend en PHP para el proyecto **Marucha**.  
En esta versión se construyó la estructura mínima operativa del backend, conectada a MySQL y preparada para exponer endpoints JSON consumibles por la aplicación móvil.

---

## Rama de desarrollo
- `feature/php-backend-base`

---

## Objetivo de la versión
Establecer la base del backend PHP del proyecto, incluyendo:

- configuración general del backend
- conexión a base de datos con PDO
- helper de respuestas JSON
- primeros modelos y controladores
- routing básico
- endpoints iniciales para consulta de categorías y productos

---
