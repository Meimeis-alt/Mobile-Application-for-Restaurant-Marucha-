# Release v0.6.0 — Cart Base

## Resumen
La versión **v0.6.0** incorpora el módulo base de **carrito de compras** para el backend PHP del proyecto **Marucha**.  
En esta versión se implementaron endpoints para consultar el carrito activo de un usuario, agregar productos al carrito, actualizar la cantidad de un ítem y eliminar productos del carrito.

---

## Rama de desarrollo
- `feature/cart-base`

---

## Objetivo de la versión
Extender el backend de Marucha para permitir que un usuario pueda:

- consultar su carrito activo
- agregar productos al carrito
- actualizar la cantidad de productos agregados
- eliminar productos del carrito

---

## Funcionalidades implementadas

## 1. Modelo de carrito
Se creó `models/Cart.php` para centralizar las operaciones relacionadas con las tablas `carrito` y `carrito_detalle`.

### Funciones implementadas
- buscar carrito activo por usuario
- crear carrito activo
- obtener ítems del carrito
- buscar ítem por carrito y platillo
- crear ítem de carrito
- actualizar ítem de carrito
- buscar ítem por ID
- eliminar ítem de carrito

---

## 2. Servicio de carrito
Se creó `services/CartService.php` para contener la lógica de negocio del módulo de carrito.

### Funciones implementadas
- obtener carrito activo por usuario
- validar existencia del usuario
- validar existencia y disponibilidad del producto
- crear carrito si el usuario aún no tiene uno activo
- agregar un producto al carrito
- sumar cantidad si el producto ya existía en el carrito
- recalcular subtotales y total general
- actualizar cantidad de un ítem del carrito
- eliminar un ítem del carrito

---

## 3. Controlador de carrito
Se creó `controllers/CartController.php` para exponer endpoints HTTP relacionados con el carrito.

### Endpoints implementados
```http
GET /api/users/{id}/cart
POST /api/users/{id}/cart/items
PUT /api/cart/items/{id}
DELETE /api/cart/items/{id}
