# Release v0.5.0 — User Profile & Addresses

## Resumen
La versión **v0.5.0** incorpora el módulo inicial de **perfil de usuario** y **gestión de direcciones** en el backend PHP del proyecto **Marucha**.  
En esta versión se implementaron endpoints para consultar y actualizar los datos básicos del usuario, así como para registrar, listar, actualizar y eliminar direcciones asociadas a un usuario.

---

## Rama de desarrollo
- `feature/user-profile-address`

---

## Objetivo de la versión
Extender el backend de Marucha para permitir que los usuarios registrados puedan:

- consultar su perfil
- actualizar sus datos básicos
- registrar direcciones de entrega
- listar sus direcciones
- editar una dirección existente
- eliminar una dirección

---

## Funcionalidades implementadas

## 1. Perfil de usuario

### Modelo de usuario
Se amplió `models/User.php` con nuevos métodos para trabajar con el perfil del usuario:

- `findById(int $id)`
- `updateProfile(int $id, array $data)`

### Servicio de usuario
Se creó `services/UserService.php` para centralizar la lógica del módulo de usuario:

- obtener usuario por ID
- validar datos mínimos del perfil
- actualizar nombre, apellido y teléfono

### Controlador de usuario
Se implementó `controllers/UserController.php` para exponer endpoints HTTP relacionados al perfil:

- obtener usuario
- actualizar usuario
